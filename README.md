# Jana's Personal Site
A minimalist, aesthetic digital portfolio showcasing my journey in STEM, robotics, and intelligent systems.

### 🔗 Try it
[Visit the live site here](https://jojy24.github.io/)

### 🚀 About
This site is a curated space where I document my technical journey. It serves as my primary portfolio for university applications, highlighting my passion for embedded systems and software development.

### ✨ Features
* **Minimalist Aesthetic:** Designed with a soft, luxury-focused UI for a better reading experience.
* **Project Showcase:** Highlights my journey from early learning to advanced engineering projects.
* **Responsive Design:** Optimized for both mobile and desktop viewing.

### 🛠 How it works
This site is built using **HTML5** and **CSS3**, utilizing **Google Fonts** for a refined typography and **GitHub Pages** for deployment. I focused on clean design patterns to ensure the content remains the focal point for any visitor or admissions officer.
